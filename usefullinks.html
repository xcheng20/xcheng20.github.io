Useful Links

Research

  • SYSDYNET: This is the webpage of the ERC ERC Advanced Research Project “Data-Driven Modeling in Dynamic Networks”.

  • Lectures on Network Systems: A nice lecture note on network systems, written by Prof. Francesco Bullo from University of California, Santa Barbara.

  • MOR WiKi: provides an overview of model reduction methods, available software implementations, and benchmarks of parametric or non-parametric models.

Toolbox and Data

  • Graph and Network Algorithms is a MATLAB embedded toolbox, which contains common graph algorithms and provides visulazation for complex networks.

  • Matpower is a package of MATLAB M-files for solving power flow and optimal power flow problems. Click here for the User's Manual.

  • Benchmark Examples for Model Reduction.

  • The Brain Connectivity Toolbox is a MATLAB toolbox for complex network analysis of structural and functional brain-connectivity data sets.

  • Network Repository is an interactive scientific network data repository.

  • SuiteSparse Matrix Collection is a large and actively growing set of sparse matrices that arise in real applications.
